## ✨ Features
* **3D Interactive Viewer:** หมุน ดู และซูมโมเดลได้แบบเรียลไทม์ผ่านหน้าเว็บ
* **Augmented Reality (AR):** รองรับการนำโมเดลออกมาวางในโลกจริงผ่านสมาร์ทโฟน
    * **Android:** รองรับผ่าน Scene Viewer และ WebXR
    * **iOS:** รองรับผ่าน Quick Look (ไฟล์ .USDZ)
* **Smooth Animation:** โมเดลมีการเคลื่อนไหว (เช่น Somchai Human Walking Model)

## 🛠️ Tech Stack
* **3D Modeling:** Blender
* **Format:** `.glb` (Android/Web), `.usdz` (iOS)
* **Frontend:** HTML5, CSS3, JavaScript
* **Library:** [Google <model-viewer>](https://modelviewer.dev/)
